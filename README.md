# Bootstrap Starter
Bootstrap Starter Template for easy Project Setup

## Dependencies
**bootstrap    : 4.3.1**
**font-awesome : 4.7.0**
**jquery       : 3.3.1**
**popper.js    : 1.14.7**

## DEV Dependencies
**browser-sync: 2.26.3**
**gulp        : 3.9.1**
**gulp-sass   : 3.1.0**


##Steps

**git clone https://github.com/avgkudey/bootstrap_starter.git**
**cd bootstrap_starter**
**npm install**
**npm start**
